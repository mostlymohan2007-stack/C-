# 🚀 Push Your Space Battle Game to GitHub

## 📋 EXACT COMMANDS TO RUN:

### **Step 1: Open Git Bash or Command Prompt**
Navigate to your project folder:
```bash
cd d:\c++\SpaceBattleGame
```

### **Step 2: Initialize Git Repository (if not already done)**
```bash
git init
git remote add origin https://github.com/Jayamurugan2220/C-.git
```

### **Step 3: Add All Files**
```bash
git add .
```

### **Step 4: Commit Your Changes**
```bash
git commit -m "🚀 Add complete Space Battle Game with SFML graphics

✨ Features:
- Real-time space combat gameplay
- Custom graphics upload support  
- Multiple enemy types (Basic, Fast, Heavy)
- Particle effects and scoring system
- Progressive difficulty scaling
- Cross-platform compatibility
- Online hosting ready

🎮 Game files:
- Complete C++ source code
- SFML graphics implementation
- HTML web hosting pages
- Console version for online play
- Comprehensive documentation

🛠️ Technical:
- Object-oriented design patterns
- AABB collision detection
- Delta time physics
- STL container optimization
- Frame rate control"
```

### **Step 5: Push to GitHub**
```bash
git push origin main
```

---

## 🎯 ALTERNATIVE: One-Liner Push Command

If you want to push everything at once:
```bash
cd d:\c++\SpaceBattleGame && git add . && git commit -m "🚀 Add Space Battle Game with SFML graphics and custom image support" && git push origin main
```

---

## 📁 What Will Be Pushed:

✅ **Complete Source Code** (src/ folder)
✅ **Header Files** (include/ folder)  
✅ **Compiled Executable** (SpaceBattleGame.exe)
✅ **Web Hosting Pages** (index.html, play_console.html)
✅ **Documentation** (README.md, guides)
✅ **Compile Scripts** (batch files)
✅ **Asset Folder** (for custom images)

## 🌐 After Pushing:

### **Enable GitHub Pages:**
1. Go to: https://github.com/Jayamurugan2220/C-/settings/pages
2. Source: "Deploy from a branch"
3. Branch: "main" 
4. Click "Save"
5. **Your game is live!** 🎉

### **Your Game URL:**
```
https://Jayamurugan2220.github.io/C-/
```
